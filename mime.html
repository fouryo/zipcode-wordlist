Microsoft IME用 単語リスト

事業所個別番号を「郵便番号+住所」に変換できるようになります。
都道府県用と市区町村用の2種類の単語リストがあります。
Microsoft IMEの辞書に一括登録してご利用ください。

都道府県用

100-8924　→　100-8924 東京都千代田区永田町

市区町村用（住所が市区町村から始まるバージョンです）

100-8924　→　100-8924 千代田区永田町

一括登録のおおまかな流れ

  1. Microsoft IMEのユーザー辞書ツールで新規ユーザー辞書の作成
  2. テキストファイルから一括登録
  3. ユーザー辞書からシステム辞書に変換
  4. プロパティでシステム辞書の追加　（忘れがち！）

※詳しくは、Microsoft IMEのヘルプなどを参照して下さい
